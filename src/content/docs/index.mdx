Welcome to your documentation site. Edit `src/content/docs/index.mdx` to
replace this page.

## Getting Started

1. Run `pnpm install` to install dependencies.
2. Run `pnpm dev` to start the local development server.
3. Run `pnpm build` to build the production site.

## Adding Pages

Create `.mdx` files inside any topic folder under `src/content/docs/`. Update
the corresponding `_meta.ts` to control sidebar order and grouping.
